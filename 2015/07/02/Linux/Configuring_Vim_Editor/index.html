Configuring Vim Editor

my vimrc

inoremap { {}kA
注:inormap是替换的命令,后面的替换前面,既用{}替换{,中间里面直接用键名字代替.kA是光标撤回的位置,以替换后的最后一排为参照点,k是向上,j是向下,和vim的操作命令一样.A是本行末尾,I是开头.
